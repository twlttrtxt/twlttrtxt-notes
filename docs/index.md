I am `twlttrtxt` (a.k.a. `Leon`), someone with a burning passion for offensive cyber-security. This page serves as a central repository for all of my notes and write-ups on various `CTF`-challenges or other small projects which i have done.

### Structure
This repository is structured as follows:
```bash
├───Active Directory
│   ├───Game of Active Directory
│   └───Setting up an AD
├───Burp Academy
├───HackTheBox-Writeups
│   ├───Machines
│   └───StartingPoint
├───Random Notes
└───TryHackMe-Notes
```
Each of these directories holds the following projects:

- `GOAD`: Write-ups on the hacking lab from [Orange-Cyberdefense](https://github.com/Orange-Cyberdefense/GOAD)
- `AD-Setup`: A documentation on the process of setting up an AD-environment from scratch. Derived from [udemy](https://www.udemy.com/).
- `Burp Academy`: Research notes from some learning paths of the [portswigger academy](https://portswigger.net/web-security).
- `HTB-Machines`: In-depth writeups on various [Hack The Box](https://www.hackthebox.com/) CTF's.
- `HTB-StartingPoint`: In-depth writeups on all starting point CTF's on [Hack The Box](https://www.hackthebox.com/).
- `Random Notes`: Simply notes on various topics that did not fit into any category.
- `TryHackMe-Notes`: Research notes from [TryHackMe](https://tryhackme.com/) modules.
